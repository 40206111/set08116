Controls for Coursework 2 Graphics Project

CAMERAS
Free Camera - default
	Activate - 0
	Move - wasd
	Reset to starting postition - c
	Reset facing direction - x

Arc Ball Camera
	Activate - 1
	Move - mouse
	Zoom - scroll wheel
	Change target island- n

LIGHTS
Directional - default on
	Toggle - l

Point - default off
	Toggle - p

Spot - default off
	Toggle - o
	Shadows - k

FILTER
Red - default 0.0
	Increase - R
	Decrease
Green - default 0.0
	Increase - G
	Decrease - Right_Shift + G
Blue - default 0.0
	Increase - RB
	Decrease - Right_Shift + B
